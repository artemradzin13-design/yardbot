# Yard Container — Telegram Bot

## Установка на Railway

1. Создай аккаунт на railway.app
2. Нажми "New Project" → "Deploy from GitHub repo"
3. Загрузи эту папку на GitHub (или используй Railway CLI)
4. В разделе "Variables" добавь три переменные:
   - TELEGRAM_TOKEN = твой токен от BotFather
   - ANTHROPIC_API_KEY = твой ключ от console.anthropic.com
   - YANDEX_GEO_KEY = твой ключ от developer.tech.yandex.ru

## Обновление базы знаний

Все тексты хранятся в переменной SYSTEM_PROMPT в файле bot.py
Чтобы добавить новую информацию — просто отредактируй этот текст и перезапусти бота.

## Что умеет бот

- Рассчитывает стоимость доставки до любого города/посёлка РФ
- Отвечает на вопросы по ценам, контейнерам, скриптам
- Выдаёт адреса и контакты терминалов
- Помнит контекст разговора (последние 10 сообщений)
